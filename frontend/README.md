# KubeWise frontend

Next.js dashboard for the KubeWise API. See the [repository root README](../README.md) for the project overview and **AWS deployment** steps.

```bash
npm install && npm run dev
```

API base URL: set `NEXT_PUBLIC_API_URL` (e.g. `http://localhost:8000/api/v1`). OpenAPI: `http://localhost:8000/docs`.
